Overview

Purpose

Site Foundry exists to help small businesses, creators, and entrepreneurs build clean, modern websites that communicate their message clearly and strengthen their online presence. The site will showcase my design services, highlight examples of past work, and guide visitors toward scheduling a consultation.

Audience

My audience includes local businesses, students, and individuals who need a simple, effective website but may not have the skills or time to build one. They value clarity, affordability, and professional presentation.

Dynamic elements

The site will include multiple JavaScript-based dynamic elements:

  • A theme switcher (light/dark mode) that uses DOM manipulation, an event listener, and conditional logic to toggle appearance and store the user's preference in localStorage.
  • A portfolio filter that uses an array of objects representing projects and filters them with array methods such as filter and map, dynamically generating project cards on the page.
  • A testimonial rotator that cycles through an array of testimonials, replacing content at timed intervals.
  • A mobile navigation toggle that listens for click events and adds/removes classes to display or hide the menu.

Content

Home page

The home page will introduce Site Foundry and explain the value of modern, clean web design. It will include a hero section, a short overview of services, a call-to-action button, and preview cards for portfolio items. Images will include a hero background graphic, icons for the service list, and a thumbnail of a sample website project.

Portfolio Page

The portfolio page will display projects dynamically using JavaScript. Projects will be stored as objects in an array and filtered using category buttons (e.g., Landing Pages, Small Business, Personal Sites). Each project card will include an image, description, and link to a live example when applicable.

Wireframes

Create two wireframes for your site. One for each page.

Home

The home wireframe will include: hero text + CTA, services section, portfolio preview, and a footer. The layout will remain responsive and collapse into a single-column layout on smaller screens.

Portfolio

The portfolio wireframe will show the filter buttons at the top, project cards in a responsive grid, and optional testimonial or CTA section at the bottom.
